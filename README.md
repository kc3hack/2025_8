## 背景・課題・解決されること
アイデア出しの期間中話し合った結果、我々関西人は関東に対抗すべきだという結論が出た...　<br>
関西は関東と比べて人口が少なく戦力差がある。しかし、情熱ではどうだろうか？ <br>
おのれの魂の「なんでやねん！！！」を用いて武力で関西をいい感じに！
